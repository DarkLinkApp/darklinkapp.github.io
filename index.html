Migration Guide

01

Backup Original Chats

Go to Settings > Chats > Chat backups. Create a backup and remember the passphrase.

02

Move the File

Locate the backup file in your file manager and move it to a safe folder (e.g. Downloads).

03

Install DarkLink

Download and install DarkLink APK. Allow installation from unknown sources.

04

Restore

Open DarkLink. Tap "Restore Backup" on the welcome screen. Select your file and enter the code.
